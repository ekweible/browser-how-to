What is an Internet Browser?

Internet browsers can be extremely useful and helpful tools for doing exactly what the name implies: browsing the Internet. This article aims to help you understand the basic features of a browser by breaking it up into four parts and describing the functions of each one. With this foundation in place, we hope that you will be able to use your browser to easily visit sites and find information.

The browser can be broken up into the following components:

  • Address Bar
  • Viewport
  • Controls
  • Tabs

If you are reading this article in a browser, you may be able to identify these components in a layout similar to the one pictured above.

The article will conclude with a list of the major Internet browsers that are available to you.

Address Bar

capabilities

  • Visiting a specific site
  • Searching for information

address

Also known as a URL, this is a unique address that points to a unique location within the Internet.

URL

URL stands for unique resource locator and is just a fancy word for an Internet address.

Every site or location on the Internet has an address, just like houses, businesses, or other physical locations. This address is how Internet users are able to navigate to a specific site. You have probably seen examples of these addresses (often called URLs) before:

  • google.com
  • nytimes.com
  • wikipedia.org

Note: addresses may be written with an http:// or www. at the beginning.

Visiting a specific site

The first function of the address bar is to visit a specific site or address. If you know the address of a site, you can simply type it into the address bar and press "enter" to navigate to that site. The content will be displayed in the viewport.

Searching for information

search engine

A search engine scours the Internet for any information that it can find and provides a tool for searching through that information.

The second function of the address bar is to search for information using a search engine like Google. This feature is useful for finding information or sites when the specific address is not known. By typing a word or a few words into the address bar and pressing "enter", a search will be performed using a search engine like Google and the results of that search will be displayed in the viewport, just like the content of a specific site would be.

Viewport

Capabilities

  • Displaying content
  • Allowing users to interact with content

link

A clickable piece of text (usually underlined or colored) that points to an Internet address. Clicking a link is the same as navigating to that address.

The viewport is in a way the simplest component of a browser because it is just a "window" to the content. However, this is also the place where the most interaction occurs. Sites will provide content for reading and links for navigating to other parts of the site or to other sites altogether, all of which occurs within this viewport.

Controls

Capabilities

  • Refreshing
  • Going backward and forward
  • Opening a new tab

Every browser has a series of controls available that accomplish a variety of tasks. These controls are located around the address bar, either above, below, or alongside it. While there are quite a few controls available, most are beyond the scope of this article.

Refreshing

Certain sites like news outlets may update their content frequently. In order to ensure that the latest content is being displayed, clicking the "refresh" button will refresh the site content.

Going backward and forward

By clicking on links that take you from one location to another, you create a "trail" of sorts that starts at the first address visited and ends at the current location. The "back" button will move one step backward in this trail while the "forward" button will move you one step forward.

Opening a new tab

As discussed in the next section, browsers have the ability to manage multiple locations or windows through the usage of tabs. New tabs can be opened by clicking the "+" button located near the list of tabs.

Tabs

Internet browsers are so useful that users often want to have multiple browsers open at one time. Tabs make this simple. The browser displays tabs at or near the top, much like a tabbed file organizer would. Each tab has its own address bar, controls, and viewport. In other words, each tab is very much like its own Internet browser, and the browser as a whole is a collection of these tabs.

Major Internet Browsers

A common misconception about browsers is that there is only one way to browse the Internet. Although personal computers come with only one Internet browser available from the start (either Internet Explorer or Safari), there are actually five major browsers available to everyone (and for free):

  • Google Chrome
  • FireFox
  • Safari
  • Internet Explorer
  • Opera
